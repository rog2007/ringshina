Установка демо скрипта:
Для установки тестового скрипта необходимо импортировать дамп таблицы (файл
podbor_shini_i_diski.sql ) в mysql базу, после чего внести в скрипт
db.php данные для доступа к mysql серверу:

$db_host     = "127.0.0.1";
$db_user     = "root";
$db_password = "";
$db_name     = 'podbor';

где $db_host адрес mysql сервера в котором находится ранее
импортированная таблица,
$db_user и $db_password реквизиты mysql пользователя
$db_name имя базы данных, в которой находится импортированная таблица.

после этого скопировать файлы index.php, 2.php, db.php, xajax.js на сервер и
запустить файл podbor.php
